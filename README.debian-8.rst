Debian 8 Jessie
===============

The basics:
::

   sudo su
   apt-get install tcsh python-virtualenv scons
   apt-get install liblinux-usermod-perl
   usermod -s /bin/tcsh vmlaker

For Matplotlib:
::

   apt-get install libpng12-dev libfreetype6-dev

For netCDF4:
::

   apt-get install libhdf5-dev
